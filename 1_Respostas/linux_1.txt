1.Por que o Linux recebeu esse nome?
	Vem do nome do criador do linux, Linus Torvald, que criou o kernel linux como uma "versão melhorada do minix", assim o nome é uma junção do nome do criador com minix.

2.O que são daemons?
	É um programa de computador que executa em plano de fundo.

3.O que é o shell?
	É um interpretador de comandos, que recebe comandos e os manda para o kernel para assim serem executados.

4.Por que é importante evitar executar o terminal como super-usuário?
	Porque caso se acesse como super-usuário, todos os comandos serão aceitos sem precisar de senha, o que pode ocasionar danos no computador, caso se utilize algum comando errado.

5.Qual botão do teclado completa o que o usuário escreve no terminal, de acordo com o contexto?
	tab
6.Quais botões do teclado apresentam instruções escritas anteriormente?
	Setas para cima e para baixo.
